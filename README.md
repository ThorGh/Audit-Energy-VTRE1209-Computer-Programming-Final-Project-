# AGV Monitoring
Developing a desktop software to calculate the electricity usage of our home appliances. The software also calculate the electricity cost for the corresponding equipment. The user can visualize the data to make it more interactive.

## General Information
Part of a Computer Programming VTRE1209 course supervised by Adlan Bagus Pradana, S.T., M.Tech. The project aim is to calculate the energy usage .
## Library Used
The library used in this project are listed below:
* PyQt5, framework for developing graphical user interfaces (GUIs).
* pyrebase, Python library that serves as a Python interface for the Firebase platform.
