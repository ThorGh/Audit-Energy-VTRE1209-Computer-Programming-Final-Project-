# Audit Energy
Developing a simple desktop software to calculate the electricity usage of our home appliances. The software also calculate the electricity cost for the corresponding equipment. The user can visualize the data to make it more interactive.
## Table of Contents
* [General Information](#general-information)
* [Library Used](#library-used)
* [Display](#display)
## General Information
Part of a Computer Programming VTRE1209 course supervised by Adlan Bagus Pradana, S.T., M.Tech. The project aim is to calculate the energy usage for each home appliance in an instantaneous manner. The software also offers data visualization of energy cost for each appliance in a bar chart format so that the user can easily notice which appliances consume largest energy.
## Library Used
The library used in this project are listed below:
* Tkinter, framework for developing graphical user interfaces (GUIs).
* Matplotlib, Python library to plot data.
